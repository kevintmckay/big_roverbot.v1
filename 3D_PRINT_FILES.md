## Printing Recommendations

| Material | Use For |
|----------|---------|
| **PETG** | Sensor mounts, motor driver mounts (heat resistance) |
| **PLA** | Low-stress brackets, cable clips |
| **TPU** | Pi 5 vibration standoffs, grommets |
| **ABS** | High-stress outdoor parts (optional) |

### Print Settings
- Layer height: 0.2mm
- Infill: 30-50% for mounts, 20% for enclosures
- Perimeters: 3-4 for structural parts
- Supports: Usually not needed for these designs
